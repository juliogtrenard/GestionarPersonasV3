# Titulo del proyecto

GestionarPersonasV3

## Descripción

Aplicación con JavaFX que guarda la información de una persona en una tabla, a partir de una ventana emergente.

## Primeros pasos

### Cómo usar

* Clona el repositorio en algún IDE y ejecuta el programa.
